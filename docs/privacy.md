BaroSit(이하 "본 서비스")은 사용자의 자세 건강을 도와드리기 위해 웹캠 영상을

(GDPR)** 의 원칙을 따릅니다.

## 2. 핵심 원칙 — 온디바이스 처리

**웹캠 영상은 사용자의 컴퓨터를 절대 떠나지 않습니다.**

- 자세 인식(MediaPipe 모델)은 100% 사용자 기기 안에서 실행됩니다.
- 본 서비스는 **카메라 영상 프레임, 캡처된 사진, 비디오 녹화를 일절 수집·전송·저장하지 않습니다.**
- 인터넷이 끊긴 상태에서도 자세 감지·점수·알림은 정상 동작합니다.

## 3. 수집·처리하는 정보의 항목

기본적으로 BaroSit은 사용자가 로그인하지 않은 경우 개인정보를 수집하지 않으며, 모든 자세 감지 및 설정은 로컬 기기 내에서 처리됩니다. 단, 사용자가 **인증 및 클라우드 동기화 서비스**를 활성화하는 경우, 서비스 제공을 위해 최소한의 정보를 수집 및 저장합니다.

### 3-1. 수집 항목 및 목적
1. **인증 및 계정 관리 (회원가입 시)**
   - **수집 항목**: 이메일 주소 (이메일 매직링크 로그인 시), 소셜 로그인 제공업체(Google, Kakao, Naver, Line, Apple 등)로부터 제공받는 고유 식별자(ID), 프로필 닉네임, 프로필 이메일, 프로필 이미지 URL
   - **수집 목적**: 회원 식별, 부정 이용 방지, 개인 맞춤 서비스 제공
2. **다중 기기 데이터 동기화 (로그인 후 명시적 동기화 시)**
   - **수집 항목**:
     - 사용자 프로필 설정: 아바타 이모지, 주 작업 환경(`work_env`)
     - 자세 분석 메타데이터: 자세 위반 이벤트 이력 (위반 종류, 지속 시간, 기기 식별자, 발생 시각 등 - **※ 실제 카메라 비디오나 사진은 절대 수집되지 않음**), 일별 점수 스냅샷(평균 점수, 위반 횟수, 스트레칭 횟수)
     - 사용자 개인 설정: 임계값, 민감도, 알림 설정 등
   - **수집 목적**: 여러 기기 간 데이터 백업 및 동기화, 주간/월간 통계 및 리포트 분석
3. **사용자 기기(localStorage)에만 저장되는 정보**
   - 아래 정보들은 사용자가 비로그인 상태일 때 로컬 기기 저장소에만 남으며, 로그인 후 클라우드 동기화를 승인하기 전까지는 서버로 전송되지 않습니다.
     - 캘리브레이션 기준 자세(머리/어깨 좌표 랜드마크 데이터 - 기기별 개별 저장)
     - 임시 자세 점수 및 설정 값들

### 3-2. 외부 전송 및 오픈 API 연동
- **MediaPipe 모델 로드**: 자세 인식을 위한 인공지능 모델 파일을 다운로드하기 위해 Google CDN(`storage.googleapis.com`)과의 통신이 발생합니다.
- **자동 업데이트 확인**: 최신 버전 갱신 여부를 확인하기 위해 GitHub API와 통신이 발생합니다.
*이러한 통신 과정에서 IP 주소 및 HTTPS 표준 헤더 정보가 인프라 수준에서 수집될 수 있으나, 본 서비스 운영자가 직접 수집하여 보관하지 않습니다.*

### 3-3. 절대 수집하지 않는 정보
BaroSit은 사용자의 프라이버시 보호를 최우선으로 합니다. **웹캠 비디오 프레임, 캡처 사진, 영상 데이터** 등 실시간 카메라 입력 정보는 로컬 기기에서 MediaPipe 모델 분석용으로 즉시 소모되며, **절대 서버로 전송되거나 디스크에 비디오/이미지 파일로 기록되지 않습니다.**

## 4. 보유 및 이용 기간

1. **기기 내 로컬 데이터 (localStorage)**: 사용자가 브라우저 캐시를 삭제하거나, 앱 내 [설정] -> [데이터 초기화]를 선택할 때까지 영구 보관됩니다.
2. **클라우드 동기화 데이터 (Supabase)**:
   - 회원 서비스 제공 기간 동안 보관하며, **회원 탈퇴 시 즉시 영구 파기**됩니다.
   - 단, 관계 법령(예: 통신비밀보호법 등)에 의거 보존할 필요가 있는 경우 해당 법령에 정한 기간 동안 분리 보관합니다.

## 5. 정보주체의 권리

사용자는 언제든지 자신의 개인정보에 대해 다음 권리를 행사할 수 있습니다:
1. **데이터 열람 및 다운로드**: 앱 내 [설정] -> [데이터 내보내기] 기능을 통해 자신이 동기화한 정보 및 로컬 데이터를 JSON 형식으로 언제든지 다운로드받을 수 있습니다.
2. **데이터 정정**: 앱 내 프로필 수정 기능 및 민감도 설정을 통해 정보를 실시간으로 수정할 수 있습니다.
3. **데이터 삭제 및 탈퇴**: 앱 내 [회원 탈퇴] 메뉴를 통해 Supabase 클라우드에 저장된 모든 데이터(프로필, 자세 점수, 위반 내역)를 **즉시 영구 삭제**하고 탈퇴할 수 있습니다. 로컬 데이터는 [데이터 초기화]를 통해 삭제할 수 있습니다.
4. **이메일 요청**: `{{ CONTACT_EMAIL }}`을 통해 개인정보 열람·정정·삭제·처리정지를 서면으로 요청하시는 경우, 운영자는 영업일 기준 7일 이내에 조치하겠습니다.

GDPR 적용 사용자(EU 거주자)는 위 권리 외에도 EU 일반 데이터 보호 규정 제15–22조에 따른 권리를 본 서비스 운영자에게 직접 요청할 수 있습니다.

## 6. 데이터 보안

- **온디바이스 처리**: 영상 데이터가 네트워크를 통해 전송되지 않으므로 외부 노출 가능성이 구조적으로 차단됩니다.
- **HTTPS**: 모든 외부 통신(자동 업데이트·MediaPipe 모델)은 HTTPS 로 암호화됩니다.
- **자동 업데이트 서명 검증**: 모든 업데이트 패키지는 minisign 서명을 검증한 뒤 설치됩니다(위변조 방지).
- **localStorage 저장 한계**: 단일 사용자 데스크탑 환경을 가정하며, OS 사용자 계정 보안 수준에 의존합니다. API 키 등 민감 정보는 향후 OS 키체인 저장으로 보강 예정입니다.

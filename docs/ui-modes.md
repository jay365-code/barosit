# UI 모드와 화면 흐름

## 모드 두 가지

### 메인 모드 (default)
큰 창에서 직접 모니터링하며 통계·설정도 함께 확인하는 평소 사용 형태.

| 요소 | 상태 |
|---|---|
| 메인 창 | 보임 |
| 미니바 (플로팅) | 보임 (설정 ON일 때) |
| 카메라 아이콘 (위젯) | 숨김 |
| 검출 owner | 메인의 `MonitorView` |

### 위젯 모드
메인 창 없이 작은 카메라 아이콘만 떠 있는 백그라운드 모드.

| 요소 | 상태 |
|---|---|
| 메인 창 | 숨김 (앱은 살아있음) |
| 미니바 | 보임 |
| 카메라 아이콘 | **보임** (미니바 옆) |
| 검출 owner | 플로팅의 `useMonitoringEngine` |

## 모드 전환 방법

### 메인 → 위젯
- 모니터 화면의 **"위젯 모드로 전환"** primary 버튼 (`switchToWidgetMode()`)
- **메인 X 클릭** — Rust가 닫기 이벤트 가로채 자동 위젯 모드 전환

### 위젯 → 메인
- **카메라 아이콘 클릭** — `switchToMainMode()`
- **트레이 아이콘 클릭** — 메인 윈도우 직접 표시 (Reopen 이벤트로 모드도 자동 전환)
- **Dock 아이콘 클릭** — 위와 동일

## 첫 실행 흐름

1. **앱 실행** → 메인 모드 + 메인 창
2. 베이스라인 없음 → **캘리브레이션 화면**
3. 5가지 적합성 체크 만족 + 시작 클릭 → 5초간 측정
4. 80% 이상 적합 시 베이스라인 저장 → **모니터 화면**
5. 자세 감지 + 점수 + 위반 표시 시작
6. 사용자가 원하면 "위젯 모드로 전환"

## 캘리브레이션 (CalibrationView)

5가지 ✓ 체크:
- 상체 보임
- 고개 들기
- 머리 좌우 수평
- 손을 얼굴에서 떼기
- 자세 유지

모두 ✓ 일 때만 "캘리브레이션 시작" 버튼 활성. 5초 측정. 적합 비율 65% 미만 시 거부.

## 메인 모니터 화면 (MonitorView)

### 평소 표시
- 카메라 영상 (또는 프라이버시 모드 시 실루엣)
- 좌상단 **점수 칩** `★ 87 pts`
- 하단 자세 chip (현재 활성 위반)
- "위젯 모드로 전환" / "다시 캘리브레이션" / "디버그 보기" 버튼

### 프라이버시 모드 (silhouette overlay)
설정에서 "실루엣 모드" 켜면 카메라 영상 대신 실루엣 렌더:
- **Image Segmenter** 마스크로 사람 영역만 색칠 (status에 따라 색)
- Face mesh 478점이 얼굴 위 점으로
- 어깨 가로 라인
- 가장자리 블러로 부드러운 글로우 (SitSense 스타일)
- 포즈 기반 클리핑으로 의자 같은 주변 사물 제거

### 디버그 패널
"디버그 보기" 버튼으로 토글 — 모든 신호 실시간 값 표시:
- person/face/hands 감지 여부
- 각 랜드마크 가시성
- 거북목 4신호 분해
- 턱괴임 5조건
- 어깨/슬라우치 점수
- 스트레칭 현재 종류

## 위젯 (플로팅 윈도우)

### 미니바 (항상 표시 — 설정 ON일 때)
- 상태색 점 + 점수 + 라벨 (좋음/주의/교정 필요/정지/자리비움)
- 드래그로 위치 이동 (저장됨)
- 단독 클릭 동작 없음 (의도치 않은 모드 전환 방지)

### 카메라 아이콘 (위젯 모드일 때만)
- 미니바 우측에 작은 원형
- 클래식 카메라 SVG (상태색)
- **클릭 → 메인 모드 복귀**

### 호버 패널
미니바·카메라 아이콘 영역 호버 시 아래로 펼쳐짐:
- 현재 감지된 자세 종류 chip
- 자세별 코칭 메시지 (예: "3분째 거북목 — 턱을 살짝 당기고 화면을 멀리")
- 마지막 알람 정보

## 일시정지 (트레이)

트레이 메뉴의 "모니터링 일시정지" / "재개" 또는 상단 일시정지 버튼.
- 점수 변동 중단
- 위반 감지 중단 (모든 추적기 reset)
- 상태 `paused`

## 자리비움

8초 이상 사람 미감지 (`ABSENCE_GRACE_MS`):
- 상태 자동 `paused` 전환
- 점수 동결
- 위젯에서는 "자리비움" 라벨로 표시 (사용자 일시정지와 구분)

## 종료

3가지 경로:
1. 설정 화면 "앱 종료" 버튼 (두 번 클릭 패턴)
2. 트레이 메뉴 "종료"
3. macOS Cmd+Q

**X 닫기 버튼은 종료가 아님** — 위젯 모드로 전환됨.

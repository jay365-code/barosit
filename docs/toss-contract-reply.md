### 2. 최대 서비스 제공기간

**최대 12개월입니다.**

- PRO 월간 구독 : 결제일로부터 1개월 단위 정기결제
- PRO 연간 구독 : 결제일로부터 12개월 (1회 선불 결제)

12개월을 초과하는 선불 상품은 취급하지 않습니다. 각 구독 주기와 다음 결제 예정일은 결제 화면 및 [프로필 > 구독 관리] 화면에 명시되어 이용자가 결제 전·후 모두 확인할 수 있습니다.

본 서비스는 물리적 배송이 없는 온라인 소프트웨어 서비스로, 결제 완료 즉시 이용이 개시됩니다.

### 3. 빌링 결제창 연동 — 완료

토스페이먼츠 빌링(자동결제) 연동을 완료하였으며, 구현 내역은 다음과 같습니다.

| 구분 | 사용 API |
|---|---|
| 카드 등록(빌링 인증) | SDK `requestBillingAuth('카드', ...)` |
| 빌링키 발급 | `POST /v1/billing/authorizations/issue` |
| 정기결제 승인 (최초/갱신) | `POST /v1/billing/{billingKey}` |
| 결제 취소·환불 | `POST /v1/payments/{paymentKey}/cancel` |

- 결제 금액은 클라이언트 값을 신뢰하지 않고 서버에서 상품 정보를 기준으로 재계산하여 승인 요청합니다.
- 발급받은 빌링키는 암호화하여 저장합니다.
- 갱신 결제는 서버 스케줄러가 1일 1회 구동하여 만기 도래 건을 청구하며, 결제 실패 시 7일의 유예기간을 부여한 후 재시도합니다.
- 웹훅을 통해 결제 상태 변경을 수신하며, 수신 건은 토스페이먼츠 조회 API로 원장을 재검증한 후 반영합니다.

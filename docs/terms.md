# BaroSit 이용약관

**최종 갱신일**: 2026년 5월 19일
**시행일**: 2026년 5월 19일

> ⚠️ 본 문서는 **초안**이며, 최종 출시 전 변호사 검토를 거쳐 확정합니다.

본 이용약관(이하 "본 약관")은 사용자(이하 "회원" 또는 "이용자")가 BaroSit(이하
"본 서비스")을 이용함에 있어 본 서비스 운영자(이하 "운영자")와 이용자 간의
권리·의무·책임 사항을 규정합니다.

---

## 제1조 (목적)

본 약관은 운영자가 제공하는 데스크톱 자세 모니터링 서비스 BaroSit 의 이용
조건과 절차, 운영자와 이용자의 권리·의무·책임 사항, 그 밖에 필요한 사항을
규정함을 목적으로 합니다.

## 제2조 (정의)

| 용어 | 정의 |
|---|---|
| **본 서비스** | 운영자가 제공하는 데스크톱 앱 BaroSit 및 관련 부속 기능 |
| **이용자** | 본 약관에 동의하고 본 서비스를 이용하는 개인 또는 법인 |
| **앱** | macOS · Windows 용 BaroSit 데스크톱 응용프로그램 |
| **자세 데이터** | 웹캠 영상의 실시간 분석으로 도출된 자세 메타데이터(좌표·점수·이벤트 등) |
| **LLM 코칭** | 이용자가 옵션으로 활성화하는 Anthropic Claude API 기반 텍스트 코칭 기능 |

## 제3조 (약관의 효력 및 변경)

1. 본 약관은 이용자가 본 서비스를 설치·실행함과 동시에 효력이 발생합니다.
2. 운영자는 합리적인 사유가 있는 경우 본 약관을 변경할 수 있으며, 변경 시 [docs/changelog.md](./changelog.md) 또는 앱 내 안내를 통해 사전 공지합니다.
3. 변경된 약관에 동의하지 않는 이용자는 본 서비스 사용을 중단하고 데이터를 삭제할 수 있습니다.

## 제4조 (서비스의 제공)

1. 본 서비스는 웹캠 영상을 사용자 기기 내에서 실시간 분석하여 자세 알림·점수·코칭 메시지를 제공합니다.
2. 본 서비스는 다음 기능을 포함합니다:
   - 자세 6종 감지(거북목·턱 괴임·어깨 기울임·등 구부정·모니터 근접·어깨 비대칭)
   - 스트레칭 7종 보너스 인식
   - 정기 휴식 알림·누적 부담 알림·자세 변동성 알림·적응형 민감도
   - 점수 시스템·대시보드·데이터 백업/복원
   - 옵션 LLM 코칭(이용자 본인 Anthropic API 키 필요)
3. 본 서비스는 무료로 제공됩니다(유료 모델 도입 시 별도 약관·동의 절차 진행).

## 제5조 (라이선스 및 지적재산권)

1. 운영자는 이용자에게 본 약관에 따라 본 서비스를 사용할 수 있는 **비독점적·양도불가능한 사용권**을 부여합니다.
2. 본 서비스의 소스 코드는 [GitHub jay365-code/barosit](https://github.com/jay365-code/barosit) 에서 공개되어 있습니다. 향후 오픈소스 라이선스가 명시될 경우 그 라이선스가 본 약관에 우선합니다.
3. 본 서비스에 포함된 상표·디자인·로고·문구의 권리는 운영자 또는 정당한 권리자에게 있습니다.
4. 본 서비스가 사용하는 제3자 컴포넌트(예: MediaPipe, Tauri, React 등)는 각 라이선스를 따릅니다.

## 제6조 (이용자의 의무)

1. 이용자는 다음 행위를 해서는 안 됩니다:
   - 본 서비스를 불법·부정한 목적으로 이용하는 행위
   - 본 서비스의 운영을 방해하거나 보안을 침해하는 행위
   - 본 서비스를 역공학·디컴파일하여 악의적으로 이용하는 행위 (단, 공개 소스코드 활용은 해당 라이선스 범위 내에서 자유)
   - 타인의 동의 없이 타인의 카메라 영상을 본 서비스로 분석하는 행위
2. 이용자는 자신이 입력한 Anthropic API 키 등 민감 정보의 관리 책임을 집니다.

## 제7조 (의료기기 아님 — 중요)

**본 서비스는 의료기기·의료기기 소프트웨어(SaMD)가 아니며, 다음과 같은 용도로
설계되지 않았습니다:**

- 질병의 진단, 치료, 경감, 예방
- 의학적 의사결정의 근거
- 물리치료·재활치료의 대체

본 서비스가 제공하는 자세 코칭·점수·알림은 **일반적인 자세 습관 형성을 돕기
위한 정보 제공**일 뿐이며, 의학적 조언이 아닙니다. 통증·불편·이상 증상이
있는 경우 반드시 의료 전문가와 상담하시기 바랍니다.

## 제8조 (책임의 제한 / 면책)

1. 운영자는 다음과 같은 사유로 발생한 손해에 대해 책임을 지지 않습니다:
   - 천재지변·전쟁·테러 등 불가항력
   - 이용자의 귀책사유(기기 손상·OS 오류·API 키 노출 등)
   - 본 서비스에 사용되는 제3자 서비스(Anthropic, GitHub, Google CDN 등)의 장애·정책 변경
   - 본 서비스 사용 결과로 의학적 결과를 기대하거나 의료 전문가 상담을 대체한 경우
2. 운영자는 본 서비스의 정상 작동을 위해 합리적 노력을 다하지만, 모든 환경(특정 웹캠 모델, 특정 OS 버전 등)에서 완벽한 동작을 보증하지 않습니다.
3. 운영자의 손해배상 책임이 인정되는 경우에도, 그 범위는 **이용자가 본 서비스 사용을 위해 운영자에게 직접 지불한 금액(현재 무료이므로 0원)** 을 초과하지 않습니다.

## 제9조 (서비스의 변경·중단)

1. 운영자는 합리적인 사유가 있는 경우 본 서비스의 기능을 변경하거나 일시 중단할 수 있습니다.
2. 본 서비스가 영구 중단되는 경우, 운영자는 최소 30일 전에 앱 내 안내 또는 GitHub Releases 공지를 통해 사전 고지합니다.
3. 서비스 중단 시에도 이용자는 자신의 기기에 저장된 데이터를 [데이터 백업 기능](./settings.md) 으로 내보낼 수 있습니다.

## 제10조 (개인정보의 처리)

본 서비스의 개인정보 처리에 관한 사항은 별도의 [개인정보 처리방침](./privacy.md)
에 따릅니다. 본 약관과 처리방침이 충돌하는 경우 처리방침이 우선합니다.

## 제11조 (분쟁 해결)

1. 본 약관에 명시되지 않은 사항은 「전자상거래 등에서의 소비자보호에 관한 법률」, 「약관의 규제에 관한 법률」 등 관계 법령과 일반 상관례를 따릅니다.
2. 본 서비스 이용으로 발생한 분쟁에 대해서는 **대한민국 법령**을 준거법으로 합니다.
3. 분쟁의 관할 법원은 민사소송법에 따른 관할 법원으로 합니다.
4. EU 거주 이용자의 경우 거주국 소비자 보호 법령상 강행규정이 우선 적용됩니다.
